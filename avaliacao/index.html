O que é Hardware?

Toda parte física do computador. Ex: Placa de vídeo, placa mãe e teclado.

O que é Software?

Todo programa presente no computador. Ex: Google, Windows e Linux.

O que é processador, memória cache, clock interno e externo?

O processador são chips responsáveis por executar instruções, decisões logicas, cálculos que tem como resultado todas as tarefas que um computador pode executar, todo processador é dividido em dois clocks: clock interno e externo. O clock interno é: velocidade de execução dos programas, e o clock externo é: Velocidade do barramento, a memória cache é pequena e muito rápida, sendo próxima do núcleo com objetivo de atender a demanda do processamento.

O que é memória RAM?

Memória volátil rápida em comparação as memórias não voláteis, utilizada pelo processador, sistema operacional e softwares para acessar rapidamente.

Conexão: DDR4.

O que é SSD (Solid State Drive)?

Dispositivos capazes de armazenar dados de forma não volátil, utiliza tecnologia de chips de memória para armazenar os dados.

Conexão: PCIe NVMe M.2
